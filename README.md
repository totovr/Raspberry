# Processing and OpenKinect

This repository contains programs for different applications in Ubuntu Xelenial and Processing 3.3.6

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

* [Raspberry Pi 3 Model B](https://www.raspberrypi.org/products/#buy-now-modal)

* Kinect V1

* Processing

### IMPORTANT

* If you did not setup the system already please check this  [README](https://github.com/totovr/Raspberry/blob/OpenKinect/README.md)
